# Biohacking methods

Biohacking in Vesprium is not one technique. Different traditions, institutions, and crises favor different ways of reshaping life. This section catalogs **methods**: how change is applied, at what scale, and what kinds of tradeoffs each path tends to create.

Subpages describe individual approaches. More will be added as the setting grows.
