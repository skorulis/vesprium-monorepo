# VespriumHackChoice

**VespriumHackChoice** is a card-based life simulator set in [Vesprium](../../docs/premise.md)—a world where biohacking, symbiosis, and quiet magic shape what people can become.

You guide a character through their life as **random cards** surface choices: events and options are dealt from the deck rather than scripted in a fixed order. Your aim is to steer that life toward **the best outcome you can**—however the cards, economy, and unfolding events allow—by picking which cards to take or which paths to skip when the rules allow it.

The app is an **iOS** Xcode project under this folder (`VespriumHackChoice.xcodeproj`). Shared Swift logic may also live in [`swift-modules/`](../../swift-modules/) at the repository root.

For setting and project context, see the [main documentation](../../docs/README.md).
