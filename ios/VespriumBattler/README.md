# VespriumBattler

**VespriumBattler** is an iOS project set in [Vesprium](../../docs/premise.md) where you monitor a bio-enhanced human during combat.

The core play loop focuses on two pressures at once:

- Keep your fighter alive through active encounters.
- Keep internal body settings under control as enhancement systems strain the body.

As fights escalate, the player must balance combat choices with ongoing regulation of biological parameters, avoiding collapse while still pushing for victory.

The app lives in this folder as an Xcode project (`VespriumBattler.xcodeproj`). Shared Swift logic may also live in [`swift-modules/`](../../swift-modules/).

For worldbuilding and project context, see the [main documentation](../../docs/README.md).
